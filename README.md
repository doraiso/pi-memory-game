# 円周率暗記ゲーム

ようこそ！
このゲームは `π`（パイ）の小数部分を1桁ずつ入力していく、シンプルだけど燃える暗記チャレンジです。

## 🎮 どんなゲーム？
- 画面に「小数点第何位か」が出るので、答えを1桁入力します。
- 正解なら次の桁へ進み、間違えるとゲーム終了。
- 連続正解した桁数がスコアになります。
- 最高記録はブラウザに保存されます。

## ✨ 特徴
- `3.` の先の `π` の小数をできるだけ長く覚えよう！
- 1問1答のテンポで遊べる。
- 「円周率入門者」から「πの住人」まで、称号で楽しめる。
- 失敗したときは正解の桁も見られるから、次の挑戦が楽しい。

## 📁 ファイル構成
- `index.html` - ゲーム画面のHTML
- `style.css` - パステル風のおしゃれなデザイン
- `script.js` - 正解判定やスコア管理などのゲームロジック

## ▶️ 遊び方
1. `index.html` をブラウザで開く
2. 出題された桁の数字を入力
3. `答える` をクリック
4. 正解なら次へ、間違えたら終了

## 💡 ローカルで動かす
1. このフォルダを開く
2. `index.html` をダブルクリックしてブラウザで開く
3. そのまま遊べます！
