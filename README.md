# 円周率暗記ゲーム

## ⭕️ 円周率 `π` の世界へようこそ！


- 画面に「小数点第〇位は？」と出るので、1桁だけ入力します。
- 1文字入力したら自動で判定され、そのまま次の桁を入力できます。
- 間違えるとゲーム終了。正解した桁数が記録になります。

## ✨ 今の特徴
- ボタンを押さずにスピーディに入力できる、直感的なUI。
- デバッグ用に好きな桁数で祝賀を表示できる仕組みも入っています。
- 小数点100桁到達では・・・？

## 📁 ファイル構成
- `index.html` - ゲーム画面のHTML
- `style.css` - ゲーム全体の見た目と祝賀演出スタイル
- `script.js` - 入力判定・スコア管理・祝賀ロジック

## ▶️ 遊び方
1. `index.html` をブラウザで開く
2. 出題された桁の数字を入力する
3. 間違えたら `もう一回` で再挑戦

## 💡 ローカルで遊ぶ
1. このフォルダを開く
2. `index.html` をダブルクリックしてブラウザで開く
